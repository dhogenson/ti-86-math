# Specks

Specks of a ti 86

Text mode: 8 lines x 21 characters
Screen res 128*64

Processor
- **CPU**: Zilog Z80 microprocessor (8-bit)
- **Clock Speed**: 6 MHz
- **Architecture**: Application Specific CPU combining Z80 core with ASIC (Application Specific Integrated Circuit)

Memory
- **Total RAM**: 128 KB
- **User-Accessible RAM**: 96 KB (the remaining 32 KB is reserved for system use)
- **Note**: All 128 KB is physically present, but only 96 KB is normally available for variables and programs
